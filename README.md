## Installation ğŸ“¦

### Installing

1. Clone the repo:

```bash
git clone https://github.com/julieoyen/semesterproject1
```

2. Navigate to the project directory:

```bash
cd semesterproject1
```

3. Install the dependencies:

```bash
npm install
```

### Running

To run the app, use the following command:

```bash
npm run start
```
